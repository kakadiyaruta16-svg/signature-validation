# ✍️ Signature Verification System

## 📌 Project Overview

The Signature Verification System is a web-based application developed using HTML, CSS, and JavaScript. It allows users to register, log in, save their signatures, and verify signatures through a simple user interface.

The system is designed to demonstrate the process of signature management and verification for educational and project purposes.

---

## 🚀 Features

- User Registration
- User Login Authentication
- Dashboard Interface
- Save Signature Functionality
- Signature Verification
- Responsive User Interface
- Easy Navigation Between Pages

## 📋 Workflow

1. User registers an account.
2. User logs into the system.
3. Dashboard page opens.
4. User can save a signature.
5. User can verify signatures.
6. Results are displayed on the verification page.
